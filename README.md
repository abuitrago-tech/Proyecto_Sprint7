# Proyecto_Sprint7
Aplicación web para listar vehículos en venta

## Descripción 
La aplicación desarrollada permite visualizar el Dataset en primer plano, realizar el desplazamiento para que el usuario tenga una vista del listado de vehiculos en venta.

## Visualizaciones
* El gráfico de barras apiladas Tipo de vehículos por fabricante cuenta cuántos vehículos hat de cada fabricante, en cada bara divide los vehículos por su tipo.
* El gráfico de Condición de vehículo permite mostral al usuario cuántos vehículos hay por modelo de año y su condicón.
* El gráfico de dispersión permite observar la relación entre dos variables numéricas. Por ejemplo el usuario pude visualizar tendencias como un vehículo con modelo más reciente el precio es más alto.
